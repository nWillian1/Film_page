# Demon Slayer: Kimetsu no Yaiba - Landing Page

Uma landing page imersiva e responsiva dedicada ao universo de Demon Slayer. O projeto foca em uma experiência visual de alto impacto, utilizando animações, trilhas de vídeo e uma estrutura de código profissional para apresentar os personagens e o trailer da obra.

**Confira o projeto online:** https://film-page-dusky.vercel.app/

---

## Funcionalidades

* **Interface Temática:** Design inspirado na estética do anime, com paleta de cores e tipografia personalizadas.
* **Seção de Personagens:** Organização modular para exibir os principais caçadores de onis.
* **Galeria de Vídeo:** Integração de trailer oficial para maior engajamento do usuário.
* **Arquitetura Escalável:** Uso de Sass com arquitetura de componentes para facilitar a manutenção.
* **Workflow Automatizado:** Processamento de assets e estilos via Gulp.

---

## Tecnologias Utilizadas

* **HTML5:** Estrutura semântica e acessível.
* **Sass (SCSS):** Estilização avançada com variáveis, mixins e separação por componentes.
* **JavaScript (ES6+):** Lógica para interatividade e manipulação do DOM.
* **Gulp:** Task runner para automação de tarefas (compilação de CSS, minificação e live reload).
* **Google Fonts:** Tipografias selecionadas (Bebas Neue, Montserrat, etc.) para compor a identidade visual.

---

##  Como rodar o projeto

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/nWillian1/Film_page.git
    ```
2.  **Instale as dependências:**
    ```bash
    npm install
    ```
3.  **Inicie o ambiente de desenvolvimento:**
    ```bash
    gulp
    ```
    *Isso abrirá o projeto no navegador e monitorará alterações nos arquivos SCSS e JS em tempo real.*
